Module: PushNotificationsWithApiKey

Used to send Push notifications via a IBM Cloud Push Notifications service with APIKey.

Parameters:
Name Type Description
bluemixRegion string

The IBM Cloud region where the Push Notifications service is hosted.
tenantId string

The tenant ID for the IBM Cloud application that the Push Notifications service is bound to.
apiKeyId string

The apiKeyId required for Push Notifications service authorization.
