LogLevel

public enum LogLevel: Int

The severity of the log message.

Used to set the logLevelFilter property.

Note

Set Logger.logLevelFilter to LogLevel.none to prohibit any logs from being recorded. Do not use LogLevel.analytics; it is only meant to be used internally by the BMSAnalytics framework.
