  • BMSURLSession

    A wrapper around Swift’s URLSession API that incorporates Bluemix Mobile Services. Use this API to gather analytics data on your network requests and/or to access backends that are protected by Mobile Client Access.

    Currently, BMSURLSession only supports URLSessionDataTask and URLSessionUploadTask.

    For more information, refer to the documentation for NSURLSession in the Swift Foundation framework.
